# API Endpoint: List of Dog Breeds

This endpoint is used to retrieve a list of breeds that can be used in various parts of the application such as search functionality, creating a new pet profile and more. The list of breeds returned by this endpoint can be used as a reference for input validation and also for providing options to the end user.

## Endpoint

`GET /api/breeds`

## Parameters

None

## Example Request

<a href="/api/breeds" className={style.link} target="_blank">
  GET /api/breeds
</a>

200 OK

## Response

- 200 OK on success.
- 404 Not Found if no breeds are available.
